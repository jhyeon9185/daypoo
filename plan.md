# 🚀 전체 서버 가동 및 로컬 환경 활성화 계획

현재 로컬 포트(5173)가 응답하지 않는 것을 확인했습니다. 모든 개발 서버를 순차적으로 기동하여 개발 환경을 활성화합니다.

## 🛠️ 작업 내용

1. **인프라 확인 (Docker)**
   - `docker-compose up -d`
   - 데이터베이스(PostgreSQL) 및 캐시(Redis) 인프라를 백그라운드에서 실행합니다.

2. **백엔드 서버 기동 (Spring Boot)**
   - `cd backend && ./gradlew bootRun`
   - 가상 스레드 기반 핵심 비즈니스 로직 서비스를 구동합니다 (Port: 8080).

3. **AI 서비스 기동 (FastAPI)**
   - `cd ai-service && python main.py`
   - 이미지 분석을 담당하는 AI 마이크로서비스를 구동합니다 (Port: 8000).

4. **프론트엔드 서버 기동 (Vite/React)**
   - `cd frontend && npm run dev`
   - 사용자 인터페이스 서비스를 구동합니다 (Port: 5173).

## 📅 작업 순서

1. `plan.md` 사용자 승인
2. Docker Compose 인프라 기동 확인
3. 백엔드 서버 백그라운드 실행 및 로그 모니터링
4. AI 서비스 백그라운드 실행 및 로그 모니터링
5. 프론트엔드 서버 백그라운드 실행 및 서비스 접속 확인
6. 최종 구동 상태 보고
